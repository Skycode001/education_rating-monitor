# 📊 Мониторинг поступающих МИФИ

Автоматический сбор и визуализация данных о поступающих.

## 🚀 Возможности

- Автоматический сбор данных с указанного сайта
- Интерактивные графики и метрики
- Отслеживание динамики изменений
- Кэширование данных для исторического анализа
- Веб-интерфейс с автоматическим обновлением
- Все настройки через .env файл
